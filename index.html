ou cliquez pour sélectionner
Choisir un fichier CSV
Formats acceptés : .csv (séparateur virgule ou point-virgule)
Format du fichier CSV attendu
nom,code_barre,categorie,fournisseur,quantite,prix_achat_ttc,prix_vente_ttc,tva,unite,seuil_alerte
Produit A,123456789,Electronique,ACME SARL,50,100,150,20,pièce,5
Produit B,987654321,Textile,,30,40,70,0,mètre,10
* Seul le champ nom est obligatoire. Les autres colonnes sont optionnelles. fournisseur : raison sociale telle qu’enregistrée dans Fournisseurs (reconnaissance sans tenir compte des majuscules) ; laisser vide si aucun. Colonnes recommandées : prix_achat_ttc et prix_vente_ttc (anciens noms prix_achat/prix_vente toujours acceptés).
Quantité : accepte 40 (remplacement), +5 (ajout), -3 (retrait) en mode Ajuster stock.
Encodage : l’import détecte UTF-8 (recommandé, avec ou sans BOM) et le CSV « Excel » classique (Windows-1252) pour que les accents (Café, à, é…) s’affichent correctement dans l’aperçu.

Import CSV — Fournisseurs

Le fichier reprend les mêmes colonnes que l’export CSV et la fiche fournisseur. Après chargement, un aperçu indique les nouveaux enregistrements, les mises à jour (même id qu’en base) et les doublons (même nom + ICE) ignorés à l’import.

ou cliquez pour sélectionner
Choisir un fichier CSV

id,nom,categorie,score,ice,if,email,telephone,ville,adresse,notes
,Fournisseur Exemple,Matériel,A,,,contact@exemple.ma,0522000000,Rabat,,
